Dúvidas Frequentes

Prezamos por uma comunicação transparente e objetiva.
Caso haja dúvidas não hesite em nos contatar!

Para que tenhamos um contato ágil e assertivo, precisamos de algumas informações específicas para fazer uma pré-análise da sua solicitação, como:

  • Qual tipo de solução você precisa? (Web, Desktop, Mobile, IOT)
  • Quais as principais funcionalidades desejadas?
  • Tem algum exemplo para ser usado como referência/base?
  • Existem regras de negócios específicas? Fluxos de informação que são chaves para seu negócio, por exemplo, uma integração com um sistema externo para análise de dados.

    • Nossa cobrança é sempre com base no serviço prestado e complexidade.
    Caso você precise de ajuda referente a tecnologia, nós pretamos auxílio como consultoria onde você é cobrado somente pelas horas consumidas nessa consultoria.
    Caso você feche um orçamento conosco, não cobramos o valor da consultoria, apenas do orçamento.
